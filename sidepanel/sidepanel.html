从采集到 AI 检索，5 分钟跑通。

这是一份内嵌教程：采集 X 内容、设置自己的模型 API、按热度排序筛选，再用 AI 对本地资料库提问。

01

采集书签和点赞

打开 X 的 Bookmarks 或 Likes 页面，点击“同步书签/同步点赞”。如果只想抓当前页面，点击“采集当前页”。

02

设置自己的 API Key

进入“设置”，选择模型服务，填写 Base URL、模型名和 API Key，先点“校验 API”，成功后保存。

03

筛选和排序

用左侧来源切换书签、点赞、转发；顶部搜索关键词、作者或标签；排序可选推荐分、点赞、转发、收藏和时间。

04

用 AI 问本地资料库

点击“AI 对话”，例如输入“找出有关设计系统的收藏”。AI 只读取本地已采集内容，回答会以卡片形式列出匹配内容。
